How to Make Everything Auto-Update
Setup guide for automated data refreshes
1. Stock Price & SEC Filings — Already live. Fetched from Yahoo Finance and EDGAR APIs on every page load.

2. Total Revenue (EDGAR XBRL) — Already live. Fetched from SEC EDGAR XBRL companyfacts API. Shows total GH revenue (not Shield-specific).

3. Google Trends — Two options:
  • Option A (recommended): Add a SerpAPI key to data.json → config.serpApiKey. Free tier: 100 searches/month.
  • Option B: Run scripts/update_data.py via GitHub Actions (uses pytrends, no API key needed).

4. SimilarWeb Traffic / Shield Page Views / Ordering Clicks — No free public API. SimilarWeb API starts at $499/mo. These must be updated manually in data.json or via the Python script with a SimilarWeb API key.

5. Shield Test Volumes / Shield Revenue / ASP — Only available in quarterly earnings. Not in any structured API. Update data.json after each earnings release.

6. Guidance / Milestones / Partners — Editorial content. Update data.json as needed.

Automated Setup: Run scripts/update_data.py on a schedule (GitHub Actions, cron, etc.) to auto-fetch Google Trends and EDGAR data, then commit the updated data.json.
Data sources: data.json, Yahoo Finance (live), SEC EDGAR (live), EDGAR XBRL (live). Web engagement metrics are modeled estimates.
